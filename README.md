# SongBird Releases

This repository hosts public binary releases for SongBird by AlphaSpring.

The SongBird application source code is private/proprietary. Do not upload
application archives into git history. macOS installers are published as
GitHub Release assets only.

## Latest Build

No public macOS build is currently available.

The v4.2.1 package was withdrawn before publication because it needs a packaging
fix. Use the release notes for the exact trust status of each future build.

## Download

Use the Releases page:

https://github.com/lokisz/SongBird/releases

## Integrity

Each release includes `checksums.txt` with SHA-256 hashes for downloadable
artifacts.
