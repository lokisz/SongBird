# SongBird Releases

This repository hosts public binary releases for SongBird by AlphaSpring.

The SongBird application source code is private/proprietary. Do not upload
application archives into git history. macOS installers are published as
GitHub Release assets only.

## Latest Build

- Version: v4.2.1
- Platform: macOS arm64
- Artifact: `SongBird-v4.2.1-mac.zip`
- Status: Developer ID signed, not notarized

Because the current v4.2.1 build is not notarized, macOS Gatekeeper may show a
warning on first launch. Use the release notes for the exact trust status of
each build.
